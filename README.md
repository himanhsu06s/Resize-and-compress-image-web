# Image Resizer and Compressor

A clean, responsive web application that allows users to upload, resize, and compress images directly in their browser. 

![Project Preview](https://github.com/himanhsu06s/Resize-and-compress-image-web/raw/main/preview.png)
## 🚀 Features

- **Instant Preview**: See your image as soon as you upload it.
- **Custom Resizing**: Manually enter width and height values.
- **Aspect Ratio Lock**: Maintain image proportions automatically while resizing.
- **Quality Compression**: Option to reduce file size for optimized web use.
- **One-Click Download**: Quickly save your resized/compressed image with a timestamped filename.
- **Clean UI**: Simple, modern design with smooth interactions.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Modern styling and responsive layout.
- **JavaScript (ES6+)**: Core logic, Canvas API for image processing, and event handling.

## 📖 How to Use

1. **Upload**: Click on the upload box or drag an image into it.
2. **Configure**:
   - Adjust the **Width** or **Height** in the input fields.
   - Use the **Lock aspect-ratio** checkbox to keep proportions.
   - Check **Reduce Quality** if you need a smaller file size.
3. **Download**: Click the **Download Image** button to save the result.

## 📂 Project Structure

- `index.html` - The main structure of the application.
- `style.css` - Custom styling and layout.
- `Script.js` - Logic for image processing, resizing, and UI interactions.
